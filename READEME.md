# Hassan Ali - Backend Developer Portfolio

## 🚀 Full-Stack Portfolio Website

A modern, responsive portfolio website showcasing my skills as a Backend Developer with React frontend and FastAPI backend.

### ✨ Features

- ✅ **Responsive Design** - Works perfectly on all devices
- ✅ **Smooth Animations** - Framer Motion animations
- ✅ **Contact Form** - Messages stored in PostgreSQL
- ✅ **Project Showcase** - Dynamic project listing from database
- ✅ **Skill Visualization** - Interactive skill bars
- ✅ **Dark/Light Mode** - Coming soon
- ✅ **Admin Panel** - Coming soon

### 🛠️ Tech Stack

**Frontend:**
- React 18
- Tailwind CSS
- Framer Motion
- React Router DOM
- Axios
- React Hook Form

**Backend:**
- FastAPI
- PostgreSQL
- SQLAlchemy ORM
- Pydantic
- Uvicorn

### 📦 Installation

#### Prerequisites
- Python 3.8+
- Node.js 16+
- PostgreSQL (or Supabase for free hosting)
